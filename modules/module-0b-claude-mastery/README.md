## How to Use This Module

### Option 1: Clone from GitHub

```bash

cd ai-native-pm/modules/module-0b-claude-mastery
claude
```

Then type `start` and follow the teacher's instructions.

Open Claude Code and type:


Claude will fetch the module and explain the setup.

## FAQ

**Q: Can I skip lessons?**
A: Not recommended. Each lesson builds on the previous one. But Lessons 5+ are somewhat independent ‚Äî you can skip to Lesson 5 (Skills) if you want to focus on that.

**Q: Do I need Claude Pro?**
A: Yes. Claude Haiku won't work for this course ‚Äî you need Pro or Max for full Claude Code access.

**Q: How long does it actually take?**
A: 2-3 hours if you work straight through. Most people spread it over 2-3 days (30 min per day).

**Q: Can I save my progress?**
A: Yes! Each lesson completion is tracked in `progress.json`. When you close and reopen the module, you'll pick up where you left off.

**Q: What if I mess up a lesson?**
A: No problem. Just delete the file or folder and start the lesson again. The teacher will guide you through.
