# Visual Identity

## Brand: The AI-Native PM

### Colors

**Primary:** Blue (#2563EB) - Trust, technology, professionalism
**Primary Light:** (#3B82F6) - CTAs, highlights
**Primary Dark:** (#1D4ED8) - Headings, emphasis

**Neutrals:**
- Gray 900: #111827 (body text)
- Gray 600: #4B5563 (secondary text)
- Gray 100: #F3F4F6 (backgrounds)
- White: #FFFFFF (cards, main background)

**Accents:**
- Green (#059669) - Free / success indicators
- Purple (#7C3AED) - Premium / paid indicators

### Typography

**Font:** Inter (Google Fonts)
- Headings: Inter Bold/Extrabold
- Body: Inter Regular
- Code: JetBrains Mono or system monospace

### Design Principles

1. **Clean over cluttered** - Generous whitespace, max 6xl container
2. **Scannable** - Use headers, bullet points, and tables
3. **Mobile-first** - Responsive at every breakpoint
4. **Fast** - Optimize images, minimal JS, static where possible
5. **Accessible** - WCAG AA contrast, keyboard navigation, alt text

### Aesthetic Reference
- Stripe (clarity, whitespace)
- Linear (modern, developer-friendly)
- Vercel (fast, clean)
