# The AI-Native PM

**Build AI-powered PM workflows in 30 minutes.**

A practical upskilling program that teaches Product Managers to use Claude Code through hands-on modules you can use immediately on your real projects.

Created by [Anmol Gupta](https://linkedin.com/in/anmol-gupta-21875a89)

## Modules

| Module | What you'll build | Time |
|--------|------------------|------|
| **Module 0: Claude Code Basics** | Get comfortable with Claude Code — install, learn 5 terminal commands, first AI conversation | 20 min |
| **Module 1: PRD Generator** | Generate production-ready PRDs with AI that asks the right questions and catches edge cases | 30 min |
| **Module 2: AI Image Generation** | Create product mockups, presentation visuals, and social media assets with DALL-E | 2 hrs |
| **Module 3: MCP Integrations Course** | Connect Claude to Jira, Google Drive, Sheets, and Figma — 7 interactive lessons | 3-4 hrs |

All modules are **free**.

## Quick Start

1. Pick a module from above
2. Open the module folder and read `QUICKSTART.md`
3. Follow the steps — you'll be productive in 30 minutes

```bash
git clone https://github.com/anmolgupta824/ai-native-pm.git
cd ai-native-pm/modules/module-1-prd
cat QUICKSTART.md
```

## Each Module Has Two Modes

- **Teaching Mode** — Interactive course with lessons, exercises, and quizzes. Claude teaches you the concepts step by step.
- **Usage Mode** — Skip the lessons. Use the AI tools directly on your real projects.

## Project Structure

```
ai-native-pm/
└── modules/
    ├── module-0-claude-basics/    # Get started with Claude Code
    ├── module-1-prd/              # PRD Generator (MCP server + 5-lesson course)
    ├── module-2-image-gen/        # AI Image Generation (MCP server + 6-lesson course)
    └── module-3-mcp-course/       # MCP Integrations (7-lesson course)
```

## Requirements

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) installed
- Node.js 18+ (for MCP servers)
- OpenAI API key (Module 2 only — for DALL-E)
